# Manual Steps

Use this reference when explaining what the user must do themselves.

## User Must Provide

- Product name and accurate product information.
- Core sell points and forbidden claims.
- Target audience.
- Reference video path or file.
- Local asset library.
- Permission to use any assets that are not self-created.
- Preferred Doubao voice and tone if not using the default.

## User Must Do In Doubao

If no Doubao API or automation is configured:

1. Paste each approved script into Doubao voice generation.
2. Select the desired voice.
3. Keep pacing close to 36 seconds.
4. Export audio as WAV or MP3.
5. Save files into `work/audio/` with matching names such as `video_01.wav`.
6. Listen once for wrong pronunciation, missing words, or awkward pauses.

## User Must Do In Jianying/CapCut

1. Import rough cuts from `work/draft/`.
2. Remove or replace bad shots.
3. Correct subtitles.
4. Adjust voiceover and music levels.
5. Add platform-appropriate stickers, transitions, captions, and cover if needed.
6. Verify product price, offer, and claim accuracy.
7. Export final videos to `final/`.

## User Must Check Before Publishing

- Copyright and asset rights.
- Douyin compliance and restricted phrases.
- No exaggerated or absolute claims.
- Accurate product information.
- Clear product visibility.
- No black frames or broken audio.
- Subtitles are readable on mobile.
- Audio and visuals are synchronized.
